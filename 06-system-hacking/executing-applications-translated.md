
# การรันแอปพลิเคชัน (Executing Applications)

- การรันโปรแกรมที่เป็นอันตรายจากระยะไกลที่ออกแบบมาเพื่อขโมยข้อมูล แคร็กรหัสผ่าน ติดตั้ง backdoor และอื่นๆ
- ขั้นตอนต่อไปหลังจากการเข้าถึงและการยกระดับสิทธิ์
- โปรแกรมที่ผู้โจมตีติดตั้งรวมถึง
  - [Backdoor](./../07-malware/malware-overview.md#backdoor) ที่ออกแบบมาเพื่อรวบรวมข้อมูลและเข้าถึงระบบโดยไม่ได้รับอนุญาต
  - Crackers ที่ออกแบบมาเพื่อแคร็กรหัสผ่าน
  - [Keylogger](#keylogger) ที่ออกแบบมาเพื่อบันทึกการกดแป้นพิมพ์
  - [Spyware](#spyware) ที่ออกแบบมาเพื่อจับภาพหน้าจอและส่งไปยังผู้โจมตี

## Keylogger

- อุปกรณ์ซอฟต์แวร์หรือฮาร์ดแวร์ที่ออกแบบมาเพื่อ
  1. บันทึกทุกการกดแป้นพิมพ์บนคีย์บอร์ดของเป้าหมาย
  2. บันทึกลงในไฟล์
  3. ส่งไปยังตำแหน่งระยะไกล
- ใช้ในการตรวจสอบกิจกรรมคอมพิวเตอร์ของพนักงานและเด็ก
- ช่วยรวบรวมข้อมูลที่เป็นความลับ รวมถึงอีเมลและรหัสผ่าน
- สองประเภท: [Hardware keylogger](#hardware-keylogger) และ [Software keylogger](#software-keylogger)

### Hardware keylogger

- ![Hardware keylogger](img/hardware-keylogger.jpg)
- ดูเหมือน USB drives และออกแบบมาเพื่อบันทึกการกดแป้นพิมพ์ซึ่งจะถูกเก็บไว้ในอุปกรณ์
- วางอยู่ระหว่างปลั๊กคีย์บอร์ดและซ็อกเก็ต USB
- ไม่สามารถตรวจจับได้โดยโปรแกรมป้องกันสปายแวร์หรือแอนติไวรัส
- สามารถค้นพบได้เพราะต้องวางไว้ทางกายภาพบนเครื่องของเป้าหมาย

#### ประเภท Hardware keylogger

- **PC/BIOS Embedded**
  - การปรับเปลี่ยนเฟิร์มแวร์ระดับ BIOS เพื่อจับการกดแป้นพิมพ์
- **Keylogger keyboard**
  - การต่อวงจรฮาร์ดแวร์กับขั้วต่อสายคีย์บอร์ด
- **External keylogger**
  - การต่อ keylogger ระหว่างคีย์บอร์ดและคอมพิวเตอร์
  - เช่น PS/2 และ USB Keylogger, Acoustic/CAM Keylogger, Bluetooth Keylogger, และ Wi-Fi Keylogger

### Software keylogger

- ติดตั้งบนเครื่องของเป้าหมาย
- การกดแป้นพิมพ์ที่บันทึกได้จะ
  - บันทึกลงในไฟล์บันทึกบนเครื่องของเป้าหมาย
  - จากนั้นส่งไปยังผู้โจมตีโดยใช้โปรโตคอลอีเมล
